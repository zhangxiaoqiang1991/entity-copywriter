# entity-copywriter：实体获客短视频脚本生成器

输入商家信息，一键生成带拍摄建议的抖音短视频脚本。覆盖餐饮/美容/汽修/教培四大行业，老板出镜拍摄。

⚠️ **发布时务必打开位置定位，这样周边人群才能刷到。**

## 使用方式

```
安装 → 采访 → 生成
```

### 安装到 AI 编程工具

| 工具 | 安装方式 |
|------|---------|
| **Claude Code** | `cp -r skills/entity-copywriter ~/.claude/skills/` 然后输入 `/entity-copywriter` |
| **Codex** | 导入 SKILL.md 作为自定义技能 |
| **OpenClaw** | 将 SKILL.md 放入 `.claude/skills/` 目录 |
| **Workbuddy** | 加载 SKILL.md 文件作为技能 |
| **Codebuddy** | 将 SKILL.md 内容粘贴为系统提示词 |
| **KimiCode** | 加载 SKILL.md 文件作为技能 |
| **Trae** | 将 SKILL.md 添加到技能目录 |
| **任意 AI 对话** | 直接把 SKILL.md 内容粘贴为系统提示词 |

### 三步使用

1. **安装** — 按上表选你的工具，装好 SKILL.md
2. **采访** — 告诉我你的情况（在哪个城市、卖什么、有什么特点）
3. **生成** — 我出 20 秒脚本 + 拍摄建议，你用剪映或开拍 APP 拍了就发

## 底层逻辑

> **你做短视频不是为了做流量，是为了做生意的。**

一条 20 秒获客视频只做两件事：

| 段落 | 时长 | 做什么 |
|------|------|--------|
| **找到客户** | 1-3秒 | 地方+品类标签精准锁客 |
| **成交客户** | 4-20秒 | 产品理由（品质/服务）+ 人设理由（实在/靠谱） |

## 核心信念

- **用流量做生意，不是做流量** — 方向对了，油门才有效
- **20 秒定生死** — 前 3 秒找客户，后 17 秒成交客户
- **发什么就会来什么** — 你想要谁看到，开头就发什么
- **老板出镜** — 对着产品说，就这么简单
- **100 条之前不谈结果** — 损失有限，回报巨大，值得疯狂干

### 🍲 餐饮 · 杭州重庆火锅

```
行业：餐饮 | 主营：重庆老火锅 | 地址：杭州西湖区 | 开店：3年 | 人均：80 | 出镜：老板
```

**选题：最近很多杭州的朋友刷到我的视频，刻意来我们家吃火锅**

| 时段 | 段落作用 | 口播文案 | 画面 | 拍摄建议 |
|------|---------|---------|------|---------|
| 1-3s | 找到客户 | 最近很多杭州的朋友刷到我的视频，刻意来我们家吃火锅 | 老板站店门口→店内环境 | 手机横屏拍门头，3秒内说完 |
| 4-7s | 成交客户 | 我们家开了3年，没打过广告，全是老客户一个带一个来的 | 店内客人吃火锅的热闹画面 | 高峰期拍，真实人气才有说服力 |
| 8-15s | 成交客户 | 用的鲜切牛肉，每天现熬底料，别的地方吃不到这个味。我做火锅就一个原则——实在，该放什么放什么，不偷工减料 | 特写牛肉→沸腾锅底→老板切菜 | 手机人像模式拍牛肉纹理，锅底沸腾冒泡时拍，暖色调 |
| 16-20s | 成交客户 | 想吃正宗重庆火锅的，左下角定位来就行，到了报我名，送你一份招牌菜 | 切门头，手指左下角 | 声音坚定，让观众觉得老板很实在 |
